## Setup

Make sure to install dependencies:

```bash
# npm
npm install

```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

```

## Start Api Server

Start the api development server on `http://localhost:2311`:

```bash
# npm
npm run api

```


## Production

Build the application for production:

```bash
# npm
npm run build

```
